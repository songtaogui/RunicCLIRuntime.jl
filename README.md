# OracliRuntime.jl

The runtime submodule for [Oracli.jl](https://github.com/songtaogui/Oracli.jl), a macro-based, strongly-typed CLI definition framework for Julia, inspired by [ArgMacros.jl](https://github.com/zachmatson/ArgMacros.jl), extended with subcommands, mutual exclusion groups, multi-value options, structured help rendering, and exception-based control flow.
